# Exercise Library Master List

Broad gym-ready exercise catalog for your app.

Fields:
- `name`
- `type`
- `muscleGroup`

Type values used:
- `weight_training`
- `bodyweight`
- `assisted_bodyweight`
- `timed`
- `cardio`
- `mobility`

Notes:
- Includes the existing default library entries already in the app.
- Equipment is included in the `name` field in parentheses.
- This is a practical master list for a commercial gym setup. It is not every possible niche variation on earth.
